Waffle grid

An easy to use flexbox grid system.

Made by @lucasgruwez

Light weight

The waffle grid system is super lightweight, weighing in at only 6kb minified using the default settings.

Responsive

With classes for medium and small devices, the waffle grid can make your layouts fully responsive.

Customisable

The waffle grid system was written entyerly in sass, so that you can easily change customize the grid to suit your needs.
